# 🚀 Meu Portfólio Pessoal - Oséias Hackbarth </>

Este projeto é um portfólio online desenvolvido como parte da **Atividade Prática da disciplina de Fundamentos da Programação Web** na **UNINTER (Ciência da Computação)**. 

Além de cumprir os requisitos acadêmicos, este site serve como minha vitrine profissional, apresentando meus projetos, formações e habilidades no ecossistema de desenvolvimento de software.

---

## 🛠️ Tecnologias Utilizadas

Para este projeto, foram utilizadas tecnologias nativas (Vanilla), sem o uso de frameworks, conforme as diretrizes acadêmicas:

* **HTML5**: Estrutura semântica do site.
* **CSS3**: Estilização, variáveis para Dark Mode e responsividade avançada.
* **JavaScript (Vanilla)**: Lógica de troca de temas, menu hambúrguer, acordeão de certificados e validação de formulário.
* **Formspree**: Integração para envio de e-mails via formulário.
* **Font Awesome**: Ícones interativos.

---

## ✨ Funcionalidades

- [x] **Dark Mode**: Persistência de tema claro/escuro via `localStorage`.
- [x] **Menu Responsivo**: Menu hambúrguer otimizado para dispositivos móveis.
- [x] **Galeria/Modal**: Zoom em imagens de projetos e certificados.
- [x] **Design Adaptativo**: Layout otimizado para telas desde 320px até resoluções superiores a 1400px.
- [x] **Validação de Formulário**: Tratamento de erros e envio real de mensagens.

## 👯 Como contribuir ou usar este projeto (Clone)

Sinta-se à vontade para clonar este repositório para fins de estudo ou para usar como base para o seu próprio portfólio!


1. **Clone o repositório:**
   ### 📖 Guia: Como Clonar este Repositório

Se você deseja ter uma cópia deste projeto na sua máquina local para estudo ou desenvolvimento, siga os passos abaixo:

#### 1. Requisito Inicial
Certifique-se de que você tem o **Git instalado** no seu computador. Para conferir, abra o terminal (ou CMD) e digite:

```bash
git --version
```
#### 2. Encontre a URL do Repositório

1. Vá até a página deste repositório no **GitHub**.
2. Clique no botão verde escrito **"<> Code"**.
3. Na aba **HTTPS**, copie o link que termina com `.git`.

> **Exemplo:** `https://github.com/usuario/projeto.git`

#### 3. Escolha onde salvar o projeto

No seu computador, abra o terminal e navegue até a pasta onde você quer que o projeto seja baixado. Use o comando **cd** (*Change Directory*):

```bash
cd Documents/MeusProjetos
```
#### 4. Execute o Comando de Clone

Agora, digite o comando **git clone** seguido pelo link que você copiou:

```bash
git clone [https://github.com/usuario/projeto.git](https://github.com/usuario/projeto.git)
```
**O que vai acontecer:**

* O Git criará uma pasta com o mesmo nome do repositório no local selecionado.
* Todos os arquivos, pastas e o histórico de versões serão baixados para dentro dessa nova pasta.

#### 5. Acesse e Use o Projeto

Entre na pasta que o Git acabou de criar:

```bash
cd nome-do-projeto
```
Agora você pode abrir essa pasta no seu editor de código (como o VS Code) e começar a trabalhar!
